This Project presents the design and implementation of a web-based Weather Analytics and Alert System developed using the Flask web framework. The system retrieves historical weather data from the Open-Meteo API, processes it using data analytics techniques, and provides insights through visualization and statistical modeling. The application integrates multiple components including API-based data acquisition, preprocessing using Pandas, visualization using Matplotlib, and machine learning using linear regression. It also detects extreme weather conditions such as high wind speeds, heavy precipitation, and temperature anomalies using predefined thresholds. Furthermore, the system enables user interaction through a query interface and supports automated email notifications via SMTP. The proposed system demonstrates how web development, data analytics, and machine learning can be combined to create an interactive and intelligent weather monitoring solution.
